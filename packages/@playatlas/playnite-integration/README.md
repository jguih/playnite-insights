# @playatlas/playnite-integration

## Purpose
Adapter module for exchanging data with Playnite.

## Dependency Direction
```text
Playnite host -> @playatlas/playnite-integration -> feature modules -> @playatlas/common
```

## Main Capabilities
- Importing game metadata from Playnite
- Synchronizing state changes into PlayAtlas
- Translating external models into internal contracts
