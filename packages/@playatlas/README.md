# PlayAtlas Packages

## Overview

```text
@playatlas
├── auth/                 # Authentication and identity layer
├── game-library/         # Game metadata management (title, playtime, tags, etc.)
├── game-note/            # Game notes domain (annotations, journaling)
├── game-session/         # Session tracking domain (open/close sessions, durations)
├── media/                # Media utilities (image handling, uploads)
├── playnite-integration/ # Adapter for Playnite API and game data sync
├── shared/               # Cross-domain utilities, types, and constants
├── sync/                 # Sync services (data exchange, cloud storage)
└── system/               # Infrastructure and bootstrap code (db, fs, logging, etc.)
```
